# Install

* Copy zarch.zip to Desktop .
* Open The Terminal .
* Type the following commands :

    - [X] cd Desktop
    - [X] unzip zarch.zip
    - [X] cd zarch
    - [X] chmod +777 zarch.sh
    - [X] ./zarch.sh  or  bash zarch.sh
